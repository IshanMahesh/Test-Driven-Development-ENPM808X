## Standard install via command-line
```bash
# Download the code:
  git clone https://github.com/vinay06vinay/Test-Driven-Development-ENPM808X.git
  cd Test-Driven-Development-ENPM808X
# Create make files
  cmake -S ./ -B build/
  # Compile and build the code
  cmake --build build/
  # Run program:
  ./build/app/shell-app

  # Run tests:
  cd build/; ctest; cd 


```
